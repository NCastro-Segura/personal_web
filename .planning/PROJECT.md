# Personal Web — Project Overview

**Owner:** Dr. Noel Castro Segura (NCastro-Segura)  
**Type:** Personal astronomy portfolio website  
**Stack:** Vanilla HTML/CSS/JS, Netlify serverless functions, ADS API  
**Deployment:** Netlify  
**Repo:** /Users/noelcastro/software/personal_web

## What This Is
A professional astronomy portfolio website for Dr. Noel Castro Segura, astrophysicist at the University of Warwick. Specializes in compact objects, time domain astronomy, accretion physics, and OPTICAM instrument development.

## Current State
- Single-page app (SPA) with 5 tabs: Home, About, Research, Publications, Contact
- Animated constellation background (40 dots + connecting lines)
- ADS integration via Netlify serverless function (Research tab shows publications)
- Dark blue/purple color scheme with rgba transparency on cards
- Standalone nbody.html simulation (not yet integrated)
- Two images in img/: agn_wind.png, SwiftJ1727_ilustration_low.jpg
